# 🌎 Projeto: Jornada Viagens

Este projeto faz parte da formação **Front-End React da Alura**, especificamente do módulo "HTML e CSS: Responsividade em Layouts".

---

## 🚀 Minha Abordagem

Embora o curso forneça o passo a passo para a reconstrução do layout, optei por desenvolver o projeto do zero de forma independente, utilizando apenas o protótipo do **Figma** como guia. Meu objetivo foi consolidar o aprendizado através da prática real e tomada de decisão técnica.

### 🛠️ Destaques do Desenvolvimento

* **Flexbox First:** Foquei no uso estratégico de Flexbox para criar um layout fluido e adaptável, reduzindo drasticamente a necessidade de múltiplas regras complexas em *Media Queries*.
* **Código Limpo (Clean Code):** Priorizei a legibilidade e a organização da estrutura para facilitar futuras manutenções.
* **Simulação de Cenário Real:** Toda a nomenclatura (variáveis, arquivos, mensagens de commit e nomes de branches) foi padronizada em **inglês**, simulando o fluxo de trabalho de uma empresa global de tecnologia.
