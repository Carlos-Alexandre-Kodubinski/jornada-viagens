# Projeto Jornada viagens
Esse projeto faz parte de um dos curso da carreira Desenvolvimento Front-End React da Alura. Especificamente "HTML e CSS: responsividade em layouts". 

### Sobre esse projeto
  Esse projeto, embora durante o curso seja ensiando como recria-lo, preferi exercitar meus conhecimentos adiquiridos até o momento do curso. Por tanto todo o projeto foi criado por mim, utilizando o projeto disponibilizado no figma pelo curso. Prefiri uma abordagem que utiliza os conceitos de flexBox para reduzir a quantidade de alterações no blocos de media query's
  Como forma de exercicio também, tentei seguir uma linha de desenvolvimento para deixar o código o mais legivel possivel. Foi utilizado o inglês para as nomeclaturas de variaveis e arquivos, bem como para os commit's e branch. tentando simular um cenário de desenvolvimento.
